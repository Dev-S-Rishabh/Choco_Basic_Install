# Windows 10 powershell script for installing some programs which are necessary for my study. This script is created by Rishabh Singh.
## Installaion is simple and easy.
### Step 1:- Right Click on Windows Button on your Homescreen as shown in the image below.
![alt text](https://github.com/Paneerdev/java-xampp/blob/main/win0.png?raw=true)
### Step 2:- Now move your cursor to Windows Powershell(admin) as shown in the image below.
![alt text](https://github.com/Paneerdev/java-xampp/blob/main/win1.png?raw=true)
### Step 3:- A simple window will open requesting you to grant admin previlages just press yes button and after that a blue screen looking window will open like in the image below.
![alt text](https://github.com/Paneerdev/java-xampp/blob/main/win2.png?raw=true)
### Step 4:- You need to just copy paste the command below to your powershell window.
``` powershell -nop -c " iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/Paneerdev/java-xampp/main/scriptimp.ps1'))" ```
### Your powershell window will look something like this.
![alt text](https://github.com/Paneerdev/java-xampp/blob/main/win3.png?raw=true)
### Step 5:- After this step let the script run it will manage to install chocolatey and will give you the option for the programs you can install.Simply type the no in front of the program name to install the program or you want to exit the script press 5.
![alt text](https://github.com/Paneerdev/java-xampp/blob/main/win6.png?raw=true)
### Step 6:- Simply install the programs you want by entering the no in front of program and the script will install the program automatically.
